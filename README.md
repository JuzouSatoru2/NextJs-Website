## Prerequisites

- npm >=6.13.7
- node >=10.14.2

## Install

```sh
npm install
```

## Usage

```sh
npm run start (or dev to develop)
```

## Usage Docker

```sh
docker-compose up --build -d
```

## Run minification script for /public files

```sh
./scripts/minify.sh or .bat
```

## Initialize the dotenv file with helper script

```sh
./scripts/init-dotenv.sh
```
