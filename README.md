## Prerequisites

- npm >=6.13.7
- node >=10.14.2

## Install

```sh
npm install
```

## Usage

```sh
npm run dev
```

## Docker

> To use whole platform including databases (Requires environment variables)

```sh
docker-compose up --build -d
```

## Run test

> Requires a started server

```sh
npm run test
```
